### 关于本项目
bilimiao原为哔哩哔哩时光机，原始功能主要为查看各个分区在每个时间段的热门视频列表，具体用法详见 [bilimiao时光机使用方法](doc/时光机.md)，后经不断改进，功能不断完善，形成了一款安卓上的[哔哩哔哩](https://www.bilibili.com/)的第三方APP。

本项目使用安卓原生技术开发，后续UI将逐步改用Jetpack Compose。
